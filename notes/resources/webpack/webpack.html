1 First init npm and install dependencies

npm init
npm install webpack --save-dev
npm install webpack-dev-server --save-dev

2 Create the config file

touch webpack.config.js

2.1 Add this to the package.json

"scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "build": "webpack-dev-server",
    "build:prod": "webpack"
}

2.2 webpack.config.js sample

var path = require ('path');  // helps to create an absolute path

module.exports = {
    entry: './js/comand.js',
    output: {
        path: path.resolve(__dirname, 'dist'),
        filename: 'bundle.js',
        publicPath: '/dist'
    }
};

2.3 Adding plugins

  • Add a package via npm install package_name --save-dev
  • Add plugins to the webpack.config.js, example:

plugins: [
    new webpack.ProvidePlugin({
        $: "jquery",
        jQuery: "jquery"
    }),
    new webpack.ProvidePlugin({
        BigPicture: "bigpicture",
        bigpicture: "bigpicture"
    })
]

2.4 Adding modules

  1. Adding modules to webpack.config.js
    • example of adding a css loader

    module: {
        rules: [
            {
                test: \.css$,
                use: [
                    'style-loader',
                    'css-loader'
                ]
            }

  2. CSS modules
    • npm install css-loader style-loader --save-dev
  3. File and URL loaders
    • npm install file-loader url-loader --save-dev
  4. Extract loader
    • npm install extract-loader extract-text-webpack html-loader html-webpack-plugin --save-dev
  5. For es2015
    • npm install babel-core babel-loader babel-preset-es2015 --save-dev
  6. For cleaning the dist
    • npm install clean-webpack-plugin --save-dev

3 Running webpack

npm run build
npm run build:prod

4 Serving with auto-refresh

Open up the app in the browser as:
http://localhost:8080/webpack-dev-server/index.html
